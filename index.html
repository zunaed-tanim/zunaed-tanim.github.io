  • Chan Chan VR

    Virtual Reality (VR) App

  • Chan Chan VR

    • Development Info

      Project Type: University Group Project
      - Darmstadt University of Applied Sciences

      Team Size: 04

      Production Time: 16 Weeks (Nov 2023 - Feb 2024)

      Tools I Used: [Unity] [C#] [Blender] [Oculus Quest 2] [SideQuest] [Trello] [G-Suite] [Miro] [Canva]

      VR App Info

      Platform: VR

      User: Single

      An interactive Virtual Reality application (vertical slice) where users can explore the ancient city of Chan Chan (Peru) through immersive and interactive experience in VR.
      We aimed to demonstrate how VR technology can be used as a learning aid for students and teaching aid for teachers to make learning about cultural heritages and history more engaging, accessible and enjoyable. We chose Chan Chan to promote relatively less known endangered cultural heritage cites.

      My Role(s)

      Developer/Programmer

      - Scripted the central event manager with coroutines and a number of other managers that handle boat movement, Chimu god timed animation, user movement limiter, corridor lights behaviour, timed portals opening-closing, user experience exit etc. using singleton system with C# in Rider and Unity.
      - Experimented with C# Events, Unity Events and Scriptable Objects for performance efficiency and modularity.
      - Set up and managed Github version control system with initally Github Desktop, later with Visual Studio Code.

      Lighting & 3D Artist

      Lighting
      Set up and managed scene lighting which including experimenting with Unity light baking features. Learned a lot about optimal application of Unity lighting options in coordination with texture setup while troubleshooting for smooth performance.

      3D Modeling
      Based on reference images found in Chimu and Chan Chan archieves,
      - Modeled multiple versions of the boat in low and mid poly count in Blender and applied initial plain gold texture(NOT the final designed texture).

      - Modeled a special sea creature that triggers a sequence of events in the experience. Learned how to rig and animate in blender to animate the sea creature model.
