  • Little Fluff

    • Development Info

      Team Size: 02

      Production Time: 2 Weeks

      Tools I Used: [Game Maker Studio 2] [GML] [Adobe Illustrator]

      Game Info

      Platform: PC

      Player: Single

      Status: Published

      Little Fluff is a mini game about a baby chicken's adventure of learning to fly (fluff!). It is a decision making platformer where the little fluff must be observent in understanding the terrain, pick the powerups and routes carefully.

      My Role(s)

      Lead Developer

      I was the only developer/programmer in the team. I educated myself of Game Maker Studio 2 and its GML framework, wrote scripts for all the mechanics including, smooth jump and wing fluff animation sequence, movements, death and spawn, powerup activation, level progression, UI intereactions and user feedbacks, set and backed up the project on github.

      Project Manager & Team Lead

      - Helped keeping the scope of the game pheasible by addressing team expertise and affordances.
      - Planned the project timeline, allocated tasks and set task goals.
      - Cooordinated team meetings and coworking sessions.
      - Resolved various development hazards by offering potential solutions
      - Created a gameplay trailer, set up the site page and published the game on itch.io.

      Lead Designer (Game, Level, UX & UI)

      - Designed 2 game levels, all the interactable terrains and its elements, powerup system and objects, jumping mechanics, floating and moving islands, traps etc.
      - Designed game UX and UI system.
      - Carried out playtesting sessions by myself and with testers and collected feedbacks. Tried to iterate on the feedbacks.

      Game & UI Artist | Animator

      - Created playable terrains, powerup object assets and traps assets.
      - Created most of the UI assets including main menu buttons, screens, powerup UI etc. (except logo and player lives).
      - Created gameplay animations for player character death and powerup glow.

      The game is available at itch.io
